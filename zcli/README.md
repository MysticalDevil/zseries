# zcli

`zcli` provides shared CLI presentation helpers for the `zseries`
workspace.

## Scope

- ANSI color styling helpers
- Help text layout helpers
- Reusable formatting primitives for terminal-oriented Zig CLIs

## Build

```bash
zig build
```

## Modules

- `src/color.zig`: color and style helpers
- `src/helpfmt.zig`: reusable help rendering helpers
